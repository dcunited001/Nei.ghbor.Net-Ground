#### Dependencies

Use apt-gem or yum to install these:

- ant1.8
- autoconf
- libssl-dev
- libexpat-dev
- libpcap-dev
- libecryptfs0
- libxml2-utils
- automake
- gawk
- gcc
- g++
- git-core
- pkg-config
- libpcre3-dev
- openjdk-6-jre-lib
